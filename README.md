# Projeto Integrador - Fase 1 e 2
## Sistema de Controle de Estoque - Garage 021

Este repositório foi criado como parte prática do Projeto Integrador da faculdade, focado em **Desenvolvimento de Sistemas** e **Empregabilidade**.

### 🚀 Sobre o Projeto
O sistema consiste em uma aplicação em formato de terminal desenvolvida em **Python** para gerenciar o estoque de mercadorias e o fluxo de faturamento de uma oficina/garagem. 

### 🛠️ Funcionalidades
*   **Cadastro de Produtos:** Permite adicionar novos itens especificando nome, quantidade inicial e preço.
*   **Listagem de Estoque:** Exibe de forma organizada os produtos e quantidades em tempo real.
*   **Módulo de Vendas:** Abate itens vendidos diretamente do estoque acumulado.
*   **Relatório Financeiro:** Apresenta o faturamento bruto gerado pelas operações.

### 💻 Tecnologias Utilizadas
*   **Python 3**
*   Estruturas de dados nativas (Dicionários e Listas)
*   Modularização de funções
